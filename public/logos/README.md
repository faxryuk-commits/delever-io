# Логотипы клиентов

Загрузите логотипы клиентов в эту папку с следующими именами:

| Клиент | Имя файла |
|--------|-----------|
| Pizza Hut | `pizza-hut.png` |
| Dodo Pizza | `dodo-pizza.png` |
| EVOS | `evos.png` |
| GIPPO | `gippo.png` |
| MAXWAY | `maxway.png` |
| Yaponamama | `yaponamama.png` |

## Требования к изображениям

- **Формат**: PNG с прозрачным фоном
- **Высота**: 80-120px (будет отображаться как 32px)
- **Ширина**: пропорциональная
- **Стиль**: Логотипы будут отображаться в серых тонах и становиться цветными при наведении

## Пример

После добавления логотипов они автоматически отобразятся в секции "Нам доверяют" на главной странице.
